# Security Policy

## Supported Versions

Only the latest major version of each `JorgeCostaMacia.*` package is supported.
If you encounter a vulnerability on an older version for which there is no
workaround, please upgrade to the latest release.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, report them privately through GitHub's
[private vulnerability reporting](https://github.com/JorgeCostaMacia/shared-net/security/advisories/new):

1. Open the **Security** tab of this repository.
2. Click **Report a vulnerability**.

You can expect an initial response as soon as possible. Once confirmed, the
issue will be fixed in the latest major version and a security advisory will be
published.
